Installation

  1. Join Potterworld and let it download the resourcepack
  2. Go to your %appdata% folder
  3. Go to .minecraft/server-resource-packs
  4. Sort the folders by date and copy the newest one
    Example of the filename: adc7daefcc6f79f28eb5708bfc7f581763b135b5
  5. Go to .minecraft/resourcepacks and paste the file you just copied
  6. Rename the copied file to Potterworld Resourcepack.zip, the icon should change
    Make sure you have File Name extensions enabled in the View tab of the File explorer Tutorial
  7. In Minecraft's server browser, select Potterworld, then Edit and change the Server resource packs to Disabled
  8. Move any seasonal or extra resource packs into the .minecraft/resourcepacks folder
  9. Done
